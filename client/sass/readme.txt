Capstone/client/sass
=============================================================================== 
	This directory contains all the SASS files for the project. These
files are generally similar enough to standard CSS to be read as such. 

    Subdirectories
    ===========================================================================   

        components
    	=======================================================================
    		Contains styles for common page elements, such as the upgrade 
    	buttons currently used for user research and location infrastructure
    	upgrades.

        globals
    	=======================================================================
        	Contains global styles applying to the application in general 
    	without reference any specific elements.

        modules
    	=======================================================================
    		Contains styles address specific page elements, typically by id.

        vendors
    	=======================================================================    	    	    	
    		Contains 3rd party style sheets.
