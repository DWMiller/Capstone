Controller: empire

This is an API endpoint which handles requests relating to user management of their assets, such as planet management and research.

Default Method: None

Methods

upgrade

Handles a request to upgrade one type of infrastructure on one planet

Arguments
  • location: Integer - the location id
  • type: String "mine/shipyard/lab" - infrastructure to upgrade
Responses
  • structure-upgrade-failure
  • user-update
  • location-update

research

Handles a request too upgrade one research category on a user

Arguments
  • type: String "armour/weapons/propulsion" - research to upgrade
Responses
  • user-update

rename

Handles a request to change the name of a planet

Arguments
  • location: Integer - the location id
  • name: String - the new name
Responses
  • location-rename-failure
  • location-update
