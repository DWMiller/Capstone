Controller: map

This is an API endpoint which handles requests for map data

Default Method

Gets user details

Responses
  • user-update

Methods

sector

Request for all data in a given sector. For this application this amounts to a request for the entirety of game map data.

Arguments
  • id: Int - id of the sector, can only be 1 at this time
Responses
  • map-update
  • user-update

system

A request for all data on a given star system

Arguments
  • id: Int - id of the system
Responses
  • map-update
  • user-update
